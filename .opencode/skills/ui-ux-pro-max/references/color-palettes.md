# UI Color Palettes by Industry

## SaaS / B2B
- Primary: Blue (#2563EB) + Slate (#64748B)
- Dark mode: Slate-900 bg (#0F172A), Blue-500 accent
- Neutral: Gray scale for content hierarchy

## E-Commerce
- Primary: Orange (#EA580C) or Emerald (#059669)
- CTA: High contrast (amber, coral)
- Trust: Blue for security badges, guarantees

## Healthcare
- Primary: Teal (#0D9488) or Blue (#0284C7)
- Clean whites, soft grays for calm feel
- Red (#DC2626) for alerts/errors only

## Fintech / Banking
- Primary: Navy (#1E3A5F) or Deep Blue (#1D4ED8)
- Green (#16A34A) for gains/positive
- Red (#DC2626) for losses/negative
- Conservative, trust-oriented palette

## Entertainment / Media
- Vibrant: Purple (#9333EA) + Pink (#EC4899)
- Dark mode default with neon accents
- Rich gradients for hero sections

## Education
- Primary: Blue (#3B82F6) + Green (#22C55E)
- Warm secondary: Amber (#F59E0B)
- Clean, approachable, friendly

## Typography System
- Display/Headings: 800 weight, tight tracking
- Body: 400 weight, 1.5 line-height
- Small/Label: 500 weight, 1.0 line-height
- Base font size: 16px (mobile minimum)
